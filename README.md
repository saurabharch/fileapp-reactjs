# Create File Sharing Web App with ReactJs, Nodejs,Mongob from Scratch.

### Installation

```
git clone https://github.com/saurabharch/fileapp-reactjs.git && cd fileapp-reactjs

```

```
cd api

```
``` 
npm install 
```

### Development

``` npm run dev ```


## Start React App

``` 
cd app 

```

```
npm start
```
