# Create File Sharing Web App with ReactJs, Nodejs,Mongob from Scratch.

### Installation

```
git clone https://github.com/tabvn/fileapp-reactjs.git fileapp

```

```
cd fileapp/api

```
``` 
npm install 
```

### Development

``` npm run dev ```


## Start React App

``` 
cd app 

```

```
npm start
```
